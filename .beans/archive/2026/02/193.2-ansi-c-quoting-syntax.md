description: |-
  Implement ANSI-C quoting $'...' syntax new in POSIX.1-2024.

  ## POSIX.1-2024 Specification
  Within $'...' sequences, backslash escapes are interpreted:
  - \a - alert (bell)
  - \b - backspace
  - \f - form feed
  - \n - newline
  - \r - carriage return
  - \t - horizontal tab
  - \v - vertical tab
  - \\ - literal backslash
  - \' - single quote
  - \" - double quote
  - \ddd - octal character
  - \xXX - hexadecimal character
  - \uXXXX - unicode (extension)

  ## Examples
  ```sh
  echo $'Hello\nWorld'     # Prints with newline
  echo $'Tab:\there'       # Prints with tab
  echo $'\x41\x42\x43'     # Prints "ABC"
  ```

  ## Implementation
  - Lexer: detect $' token sequence
  - Parser: handle as special string literal
  - Process escape sequences during lexing/parsing
  - Integrate with existing string handling

  ## Files to Modify
  - src/lexer/mod.rs - add AnsiCString token
  - src/parser/mod.rs - parse $'...' sequences
  - src/executor/mod.rs - no changes (handled at parse time)
