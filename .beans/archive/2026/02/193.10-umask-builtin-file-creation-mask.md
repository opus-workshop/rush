description: |-
  Implement the umask builtin for file creation permission control.

  ## POSIX Specification
  umask [-S] [mask]

  - No args: print current umask (octal)
  - -S: print in symbolic form (u=rwx,g=rx,o=rx)
  - mask: set new umask (octal or symbolic)

  ## Implementation
  - Call libc umask() to get/set
  - Parse octal notation (0022)
  - Parse symbolic notation (u=rwx,g=rx,o=rx)
  - Format output for display

  ## Files to Modify
  - src/builtins/umask.rs - new file
  - src/builtins/mod.rs - register builtin
